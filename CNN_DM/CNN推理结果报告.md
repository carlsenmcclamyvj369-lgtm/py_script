# CNN Mosquito Noise 推理结果报告

## 1. 概述

使用 4 层 CNN 模型对 32 张测试图片进行蚊式噪声 (Mosquito Noise / Gibbs Ringing) 检测。模型将每张图划分为 8×8 block，对每个有效 block（排除图像边界 4 个 block 的 CNN 感受野不足区域）输出 0~1 的概率值，以 0.5 为阈值判定是否为蚊式噪声。

## 2. 模型信息

| 项目 | 值 |
|------|-----|
| 模型文件 | `mosquito_denoise_cnn.pth` (137.8 KB) |
| 网络结构 | 4 层 Conv2D + BatchNorm + Sigmoid |
| 层配置 | 16→32→64→16→1, kernel=3, padding=0 |
| 输入特征 | 16 维 (方差、边缘、振铃、二阶差分等) |
| 输入大小 | 9×9 邻域 patch (16通道) |
| 训练数据 | DM: 9,960 patches (806,760 行), Not DM: 10,412 patches (843,372 行) |
| 训练/验证切分 | 8:2 分层随机切分（每类各自切分，保持正负比例） |
| 训练集 | DM: 7,968 patches, Not DM: 8,330 patches, 共计 16,298 |
| 验证集 | DM: 1,992 patches, Not DM: 2,082 patches, 共计 4,074 |
| 损失函数 | BCELoss |
| 优化器 | Adam, lr=1e-3 |
| 训练轮数 | 40 epochs |
| 模型保存策略 | 按验证集 F1 保存最佳模型 |
| 最佳验证 F1 | **0.9997** (Epoch 37) |
| 最佳验证准确率 | **99.98%** |
| 最佳混淆矩阵 | TN=2,081, FP=1, FN=0, TP=1,992 |

> 训练和推理均在 CPU 上运行。每张 1080p 图片推理平均耗时约 40~60 秒。

## 3. 训练流程

### 3.1 数据切分

采用分层随机切分，确保训练集和验证集中正负样本比例与原始数据一致：

- 先将 DM (9,960 patches) 和 Not DM (10,412 patches) 各自内部随机打乱
- 各取 20% 作为验证集，80% 作为训练集
- 合并两类的训练集和验证集，保证正负样本均衡

### 3.2 训练过程

每轮训练完成后，在验证集上计算完整指标：

| 指标 | 说明 |
|------|------|
| Loss | 训练集上的 BCELoss |
| Accuracy | 验证集预测准确率 |
| Precision | 精确率 = TP / (TP + FP) |
| Recall | 召回率 = TP / (TP + FN) |
| F1 Score | F1 = 2 × P × R / (P + R)，作为模型保存依据 |
| Confusion Matrix | TN, FP, FN, TP 四象限 |

模型仅在验证集 F1 创新高时保存，最终保留的是 Epoch 37 的最佳模型（F1=0.9997）。

### 3.3 训练过程摘要

Epoch 1~10：Loss 从 0.137 快速下降到 0.002，F1 快速提升至 0.99+
Epoch 11~20：Loss 稳定在 0.001 以下，F1 在 0.997~0.999 之间波动
Epoch 21~30：偶见 Loss 反弹 (如 Epoch 21 Loss=0.0099, F1=0.953)，但很快恢复
Epoch 31~40：持续小幅优化，Epoch 37 达到最佳 F1=0.9997（TN=2081, FP=1, FN=0, TP=1992）
最终 Epoch 40：F1=0.9992（TN=2079, FP=3, FN=0, TP=1992）


## 3. 推理流程

1. 读取 BMP 图像，转为 BT.709 Y 亮度通道
2. 按 8×8 grid 计算 49 维特征图
3. 从中选取与训练一致的 16 个特征，按固定除数归一化到 [0,1]
4. 对每个非边界 block，提取其 9×9 邻域特征 (81×16)，reshape 为 (16,9,9) 张量
5. CNN 推理输出概率，> 0.5 判定为 DM
6. 在原图上叠加红色标记 DM block，绘制灰色网格线

## 4. 16 个输入特征

| 序号 | 特征名 | 说明 | 归一化除数 |
|------|--------|------|-----------|
| 1 | mean_var | 3×3 窗口方差均值 | 1020 |
| 2 | low_var_count | 低方差像素计数 (<100) | 64 |
| 3 | high_var_count | 高方差像素计数 (>500) | 64 |
| 4 | edge_strength | 边缘强度 (水平/垂直梯度均值较大者) | 255 |
| 5 | edge_orientation_conf | 边缘方向置信度 | 1.0 |
| 6 | col_ringing_mean | 列方向振铃评分均值 | 1.0 |
| 7 | row_ringing_mean | 行方向振铃评分均值 | 1.0 |
| 8 | second_diff_max | 二阶差分能量 (行列较大者) | 510 |
| 9 | second_diff_min_max | 二阶差分行列比值 | 1.0 |
| 10 | profile_ringing_mean | 所有轮廓振铃评分均值 | 1.0 |
| 11 | ringing_mean_min | 行列振铃均值中较小者 | 1.0 |
| 12 | ringing_mean_min_max | 行列振铃均值比值 | 1.0 |
| 13 | row_ringing_max | 行方向振铃评分最大值 | 1.0 |
| 14 | row_diff_max | 行均值差最大值 | 255 |
| 15 | col_ringing_max | 列方向振铃评分最大值 | 1.0 |
| 16 | col_diff_max | 列均值差最大值 | 255 |

## 7. 使用方式

```bash
# 训练模型
python dm_cnn.py

# 单张图片推理
python predict_cnn.py <bmp_path>

# 批量处理 test_data 下所有 BMP
python predict_cnn.py --batch <test_data_dir>
```
